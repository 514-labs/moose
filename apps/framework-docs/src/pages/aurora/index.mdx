# Aurora Research Preview

Aurora is your data engineering AI team. It is an agentic data engineering [MCP](https://modelcontextprotocol.io/introduction) that has your code, data, and infrastructure in context. Will help you build and build with your data systems. Won't get in your way.

Supports [Claude Desktop](https://claude.ai/download) and [Cursor](https://www.cursor.com/). May work with other hosts.

Install Moose (Fiveonefour's open source developer framework for Data Engineering) and Aurora:

```
bash -i <(curl -fsSL https://fiveonefour.com/install.sh) aurora,moose
```

### Requirements

- Requires an [Anthropic token](https://www.anthropic.com/), as some tools are agentic.
- Requires [Docker](https://www.docker.com/) and [Node](https://nodejs.org/en) for Typescript use.  
- Requires [Docker](https://www.docker.com/) and [Python 3.12+](https://www.python.org/downloads/) for Python use. 
- Some templates have other dependencies.
