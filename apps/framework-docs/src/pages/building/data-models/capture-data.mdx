# Capture Data
MooseJS makes it easy to capture data from your user-facing applications by providing typed SDKs that are automatically generated from your data models. 

## From MooseJS SDKs
<Callout type="info" emoji="ℹ️">
  We currently support Typescript SDKs. Support for other languages is coming soon.  If you'd like to see a particular language supported, please join our [Slack community](https://join.slack.com/t/moose-community/shared_invite/zt-2fjh5n3wz-cnOmM9Xe9DYAgQrNu8xKxg) and let us know.
</Callout>

You can develop your data service right alongside your user-facing applications. MooseJS will automatically generate a Typescript SDK as you build your service locally and make it available for you to link to your application. To link a data service's SDK to your application, simply run the following command:

```txt filename="Terminal"
// From your application's root directory or wherever you run npm install
npm link <project-name>-sdk 
```

## Through Ingestion Points
As you define data models in your service, MooseJS will automatically generate ingestion points for you to send data to. Ingestion points are HTTP endpoints that you can send data to.

<Tabs items={['curl', 'javascript', 'python']}>
  <Tabs.Tab>
    ``` bash
  curl -X POST \
    -H "Content-Type: application/json" \
    -d '{"eventId": "1234567890", "timestamp": "2019-01-01 00:00:01", "userId": "123456", "activityType": "click"}' \
    http://localhost:4000/ingest/UserActivity
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```javascript
  fetch('http://localhost:4000/ingest/UserActivity', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      eventId: '1234567890',
      timestamp: '2019-01-01 00:00:01',
      userId: '123456',
      activityType: 'click'
    })
  })
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```python
    import requests

    requests.post('http://localhost:4000/ingest/UserActivity', json={
      'eventId': '1234567890',
      'timestamp': '2019-01-01 00:00:01', 
      'userId': '123456', 
      'activityType': 'click'
    })
    ```
  </Tabs.Tab>
</Tabs>
