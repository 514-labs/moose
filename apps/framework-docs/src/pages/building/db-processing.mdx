# Introduction to Blocks

Once your data is in their respective tables in the database, you might want to aggregate it into Views or Materialized Views to pre-compute metrics, time series, or other computationally expensive data analysis tasks.

The following section walks through how to leverage Moose **Blocks** to create Views or Materialized Views of your database tables, where you define SQL SELECT statements to populate these views with the query result set.
