## Overview
The system takes raw aircraft tracking data (such as ADS-B data) and transforms it into a flattened data model that's optimized for storage and querying.

It is currently only pulling from military aircraft.

## Project Structure

### Data Models

| Type | Name | Description |
| ---- | ---- | ----------- |
| **Raw** | `AircraftTrackingData_altBaroString` | Raw aircraft data where altitude may be represented as a string (e.g., "Ground" or numeric values) |
| **Processed** | `FlattenedAircraftTrackingData` | Standardized data model with normalized fields and additional computed values |

### Processing Functions

| **Input Model** | **Output Model** | **Description** |
| -------- | -------- | --------------- |
| `AircraftTrackingData_altBaroString` | `FlattenedAircraftTrackingData` | Transforms the raw data into the flattened format, handling special cases like "Ground" altitude |
