# Configuring Data Models

Each data model's configuration can be tweaked to fit your needs.
To set the configuration for a data model, just export a configuration object from the model file
with the data model name as a prefix and `config` as the suffix, like so:

```ts filename="UserActivity.ts" copy
// <modelName>Config
export const UserActivityConfig = {
  ingestion: {
    format: "json",
  },
};

type Key<T extends string | number> = T;

export interface UserActivity {
  eventId: Key<string>;
  timestamp: string;
  userId: string;
  activity: string;
}
```

## Configuration Options

### Ingestion

Configures the API endpoint that we provision to ingest data into the data model.

```ts copy
export const UserActivityConfig = {
  ingestion: {},
};
```

#### Format

The format of the data that the ingestion endpoint will accept.

```ts copy
export const UserActivityConfig = {
  ingestion: {
    format: "json",
  },
};
```

Possible values:

- `json`: Single JSON object expected in the body of the request
- `jsonArray` Array of JSON objects expected in the body of the request
